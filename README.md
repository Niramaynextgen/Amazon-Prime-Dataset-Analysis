# Amazon Prime Dataset Analysis

## Overview
The Amazon Prime Dataset Analysis project is an in-depth exploration of Amazon Prime Video's content library. The analysis aims to uncover insights regarding viewership trends, genre distribution, and other key metrics that can help stakeholders understand the landscape of streaming content on Amazon Prime.

## Objectives
- Analyze the distribution of content across various genres.
- Explore historical trends in viewership and ratings.
- Provide visualizations to help stakeholders make data-driven decisions.

## Dataset Description
This project utilizes the Amazon Prime Video dataset, which includes information such as:
- Title
- Genre
- Release Year
- Rating
- Availability
- View Count

## Tools and Technologies
- **Programming Language**: Python
- **Data Analysis Libraries**: Pandas, NumPy
- **Visualization Tools**: Matplotlib, Seaborn
- **Jupyter Notebook**: For interactive analysis and visualization.
